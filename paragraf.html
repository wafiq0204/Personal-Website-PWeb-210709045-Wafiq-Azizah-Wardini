HTML

HTML singkatan dari Hypertext Markup Language adalah bahasa markup yang digunakan untuk membuat halaman web. HTML memungkinkan pengguna untuk membuat struktur dan konten dari sebuah halaman web dengan menggunakan berbagai elemen seperti judul, paragraf, gambar, tautan, tabel, dan lainnya. Setiap elemen HTML ditandai dengan tag yang memberikan instruksi kepada browser tentang bagaimana cara menampilkan konten tersebut kepada pengguna. Dengan HTML, pengguna dapat membuat halaman web yang interaktif, dinamis, dan mudah dinavigasi. Selain itu, HTML juga bekerja sama dengan CSS (Cascading Style Sheets) dan JavaScript untuk memperindah tampilan halaman web dan menambahkan fungsionalitas yang lebih kompleks.


CSS

CSS merupakan singkatan dari Cascading Style Sheets adalah bahasa gaya yang digunakan untuk mengontrol tampilan dan tata letak dari halaman web yang ditulis dalam HTML. Dengan CSS, pengguna dapat menentukan berbagai properti seperti warna, ukuran teks, jenis font, jarak antar elemen, dan banyak lagi untuk setiap elemen dalam halaman web. CSS memungkinkan pengguna untuk memisahkan struktur dan konten dari desain visual, sehingga memudahkan untuk mengubah tampilan halaman web secara konsisten di seluruh situs. Selain itu, CSS juga mendukung konsep pewarisan, di mana properti gaya yang didefinisikan untuk suatu elemen juga dapat diterapkan pada elemen yang lebih spesifik di dalamnya, sehingga menghemat waktu dan upaya dalam pengembangan halaman web yang kompleks.
